This Privacy Policy describes how Calmkeep collects, uses, and protects information.

1. Information We Collect

We collect only the minimum information necessary to operate the service:

  • Email address (via Stripe checkout)
  • Subscription status
  • Calmkeep API key
  • Basic usage metadata (request counts, latency metrics)
  • Billing information processed by Stripe

2. Information We Do Not Collect

  • We do not store user prompts
  • We do not store model responses
  • We do not store conversation logs
  • We do not persist Claude API keys
  • We do not inspect model content beyond runtime processing

Calmkeep does not retain LLM content. Prompt and response data pass through runtime execution but are not stored.

3. Third-Party Services

Calmkeep relies on the following third-party providers:

  • Stripe (payment processing)
  • Supabase (subscription storage)
  • Modal (hosting infrastructure)
  • Anthropic (user-provided API integration)

Each third-party provider maintains its own privacy and data practices.

4. Data Retention

Subscription and billing records are retained as required for account management, billing verification, and compliance obligations.

Calmkeep does not retain prompt or response content.

5. Security

We use reasonable technical safeguards to protect stored subscription data. However, no system can guarantee absolute security.

6. Your Rights

You may:

  • Cancel your subscription at any time
  • Request deletion of subscription-related data
