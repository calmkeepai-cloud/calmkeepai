Calmkeep Documentation

Getting started with the Calmkeep continuity layer for Claude.

Quick Start

Install the Calmkeep SDK and route your Claude API usage through the Calmkeep runtime.

pip install calmkeep

Then initialize the client:

from calmkeep import CalmkeepClient

client = CalmkeepClient(
    calmkeep_key="YOUR_CALMKEEP_SUBSCRIPTION_KEY",
    claude_key="YOUR_CLAUDE_API_KEY"
)

response = client.complete(
    prompt="Continue evolving the existing multi-tenant API..."
)

Calmkeep preserves structural continuity across extended multi-turn workflows.

Claude Code Plugin

Developers using Claude Code can install the Calmkeep plugin directly.

/plugin marketplace add calmkeepai-cloud/calmkeep-claude-plugin
/plugin install calmkeep-continuity

After installation, set your Calmkeep API key:

export CALMKEEP_API_KEY=YOUR_CALMKEEP_SUBSCRIPTION_KEY

Plugin repository:
https://github.com/calmkeepai-cloud/calmkeep-claude-plugin

Using Calmkeep in Cursor

Cursor users can integrate Calmkeep by routing Claude calls through the SDK.

pip install calmkeep

Example integration:

from calmkeep import CalmkeepClient
import os

client = CalmkeepClient(
    calmkeep_key=os.getenv("CALMKEEP_API_KEY"),
    claude_key=os.getenv("ANTHROPIC_API_KEY")
)

Calmkeep ensures architectural decisions remain consistent across long Cursor sessions.

Using Calmkeep with OpenClaw

OpenClaw workflows can integrate Calmkeep through environment variables and the SDK.

export CALMKEEP_API_KEY=your_key
export ANTHROPIC_API_KEY=your_claude_key

Then call the Calmkeep client within your workflow scripts.

Calmkeep acts as a runtime orchestration layer around Claude API usage.

Architecture Overview

Your Application → Calmkeep Runtime → Claude API → Structured Response

  • Bring your own Claude API key
  • Claude billing remains directly with Anthropic
  • No token resale or proxy billing
  • No modification of model weights
  • External runtime orchestration only

Troubleshooting

Missing Calmkeep API key

export CALMKEEP_API_KEY=your_key

Claude API key not configured

export ANTHROPIC_API_KEY=your_claude_key

Ensure both keys are available to your environment before running Calmkeep.
